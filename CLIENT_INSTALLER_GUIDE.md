# KODYS Installation Guide for Clients

Thank you for choosing **Kodys Foot Clinik**. To ensure a smooth installation of our medical suite, please follow these simple steps for Windows 10/11.

## 📥 Installation Steps

1.  **Download the Installer**: 
    - [Download KODYS v2.2.5 Final Gold Release](https://github.com/pearlraja30/kody/releases/latest)

2.  **Run the Installer**:
    - Launch the downloaded `Kodys_Foot_Clinik_v2.2.5.exe`.
    - **Note on Windows SmartScreen**: If a blue box appears saying **"Windows protected your PC"**, simply click **"More info"** and then **"Run anyway"**.

3.  **Direct-Install**:
    - The setup will now perform a **Direct-Install**. No extra windows or "Next" clicks are required. It will install the medical suite and create your desktop icon automatically.

4.  **Automatic Launch**:
    - Once finished, KODYS will launch automatically.
